# Security Policy

## Supported Versions

Currently, only the latest commit on the `main` branch is officially supported with security updates. We recommend users always run the latest version.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Security is a high priority for us. If you discover a security vulnerability within this project, please **do not** report it by creating a public GitHub issue. 

Instead, please privately report it directly to the repository maintainers by email or through GitHub's Private Vulnerability Reporting feature (if enabled on the repository).

We will try to acknowledge receipt of the vulnerability report within 48 hours and provide a timeline for triage and resolution.

Once the vulnerability is confirmed, we will work on a patch and release an update as quickly as possible. We will also credit you for the discovery in the release notes if you wish.

Thank you for helping us keep this project safe!
