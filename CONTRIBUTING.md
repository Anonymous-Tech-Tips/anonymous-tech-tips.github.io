# Contributing to Anonymous Tech Tips

First off, thank you for considering contributing to Anonymous Tech Tips! It's people like you that make this community great.

## Where to start?

1. check out [the open issues](https://github.com/Anonymous-Tech-Tips/anonymous-tech-tips.github.io/issues)
2. Pick an issue you would like to work on and leave a comment.
3. If you have an idea for a new feature or game to add, please create an issue first to discuss it before doing the work.

## Development Workflow

1. Fork the repo and create your branch from `main`.
2. Run `npm install` in the repository root.
3. If you've added code that should be tested, add tests.
4. If you've changed APIs or features, update the documentation.
5. Ensure the test suite passes.
6. Issue that pull request!

## Guidelines for adding Games

We love adding new, high-quality, unblocked web games. When submitting a PR to add a game, ensure:
- The game is hosted securely and reliably.
- We have the rights or permission to embed/link it.
- You provide a high-quality thumbnail (16:9 or 4:3 preferred) in the `public/games/img` directory.
- You insert the game details directly into the `src/data/games.ts` array.

## Pull Request Process

1. Provide a comprehensive summary of your changes in the Pull Request description.
2. Link the PR to the relevant issue (e.g. `Fixes #123`).
3. Maintain clean commit history where possible.

## Code Style

- We use standard TypeScript/React conventions. 
- Please run any formatters or linters the project has configured before submitting.

Thank you for contributing!
